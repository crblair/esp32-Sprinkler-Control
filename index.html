Features

  • Manual zone control: Turn individual zones ON/OFF from the web UI
  • Quick Run Mode: Start all enabled zones for a set time, with live countdown and visual highlights
  • Web-based interface: Works on desktop and mobile
  • Live status updates: Visual highlights for active Quick Run
  • Easy configuration and scheduling: (Optional) Set up automatic schedules for watering
